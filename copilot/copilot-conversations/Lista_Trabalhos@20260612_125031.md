### Como usar estes templates:

1.  **Instala o plugin Templater:** Se ainda não o tens.
2.  **Cria uma pasta para templates:** Ex: `_Templates`.
3.  **Copia e cola** o conteúdo de cada template para um novo ficheiro `.md` dentro dessa pasta (ex: `Template - Obra.md`, `Template - Lote.md`).
4.  **Configura o Templater:** Nas definições do plugin, aponta para a tua pasta de templates.
5.  **Cria novas notas:** Usa o comando "Templater: Insert template" ou configura hotkeys para criar notas a partir destes templates.
